This folder is intended to hold 3D models used by the game.

Place your converted Montpellier district models here.  By
default the program looks for a file named `example.glb` at runtime,
but you can pass any file path as a command‑line argument when
running the executable.

Raylib supports several formats including Wavefront OBJ, glTF/GLB,
IQM, Vox and M3D【99204103476631†L2368-L2374】.

If you convert a DXF file to GLB using `assimp` or Blender, copying
it into this folder will make it available to the application.
